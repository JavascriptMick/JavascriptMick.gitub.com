ReLearn attempts to solve the problem of revising learned material by prompting users to revise at an optimum time for recall. A work in progress, I am implementing a web based portal in Java, Spring MVC and AngularJS with plans for native mobile apps for Android (java) and IOS (swift).

This Portfolio is implemented using Reveal.js and AngularJS. I wrote a custom directive to allow the creation of the key menu and also published the directive to NGModules.Org.

StrongSheet.com is an exercise in lean startup and internet marketing. I used Lean Canvas to create business plans, customised an html template, created an introduction video using PowToon and a click-through wireframe using Balsamiq Mockups.

Hackify.org allows developers to instantly take code from their local machine and share it to an online collaborative IDE. I implemented the solution using Node.js, Redis, Socket.io and AngularJS. I self hosted the solution on digital ocean with multiple Forever instances load balanced with nginx. Hackify is 100% Open Source.

During lean business modeling, I identified many use cases for Hackify. One of the more interesting is using it to get help on programming problems. This use case is illustrated in the video.

GuideBySide provide interactive guides for online customers. It was implemented by porting my C# RuleClarity engine to javascript/Node.js and introducing AngularJS and Socket.io for the web client and real time communications respectively.

I converted the GuideBySide widget into a Shopify App. It allowed shop owners to embed an interactive guide into their own shop. Implementation was in Node.js, Express, MongoDB and AngularJS.

Packaging the angular application as a widget with grunt was particularly interesting, in fact so interesting I made a video about it.

The guide builder is an interactive rule builder with a 'drag and snap' user experience inspired by the scratch programming tool. I implemented this in AngularJS.

Spodgic is a real time, multiplayer geoactive RPG available for Android devices on Google Play. I implemented as a Mobile HTML5 solution using PhoneGap, JQuery Mobile, Node.js and Google Maps.

Google Maps integration allows the user to use their current location as a 'battleground', placing factory units and monsters.

I leveraged a Javascript based gesture library over HTML5 Canvas to enable Gesture based combat.

RuleClarity is a Rules engine with an english language syntax. Implemented in C#, it includes a core rule evaluation engine, a backend for storing and managing rules and an invocation API. In addition, I implemented an integrated 'scratch' style drag-and-connect rule builder interface using Winforms.

The rules engine was built on a technology that I created and termed 'Semantic Rules' which I explain in this (rather long) video.

SMS Holdem allows you to play a game of Texas Holdem poker with other players by sending and receiving SMS messages. I implemented the solution including my own poker engine in C#, ASP.Net, Joomla and MySQL

Zilch is a multiplayer dice game implemented in C# and ASP.Net.

PoweredByUsers is a content management and site generation solution designed to allow non technical users to update and publish a website. The site was data driven and extensible with modules like News and file downloads. I implemented the solution in C#, ASP.Net and XSLT.
